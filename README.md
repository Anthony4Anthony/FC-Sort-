# FC-Sort

This is a simple, straight forward algorithm called FC-Sort, and implemented in Python programming language. It is a Frequency Count - Distribution based sort algorithm that is competitive with leading algorithms in sorting Low Entropy datasets especially those for which the number of distinct (unique keys) is far less than the dataset size n.



\# FC-Sort — Frequency Count Sort



A near-linear time distribution-based sorting algorithm designed for

low-entropy, duplicate-heavy datasets where u << n.
